# Systém pro evidenci revizí spalinových cest

Tento projekt je prototypem webové aplikace pro revizní techniky, inspirovaný systémem **CESC** (Centrální evidence spalinových cest).

### Hlavní funkce:
- [cite_start]**Digitální formulář:** Sběr dat o spotřebičích (např. Luna Compact 24) a kouřovodech[cite: 18, 43, 46].
- [cite_start]**QR Ověření:** Automatické generování QR kódu pro okamžité ověření pravosti zprávy online[cite: 77].
- [cite_start]**Export do PDF:** Formátování zprávy připravené k tisku dle normy EN 14471[cite: 56].
